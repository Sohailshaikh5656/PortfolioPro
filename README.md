## 📥 Installation Steps

Follow these steps to set up the project locally:

### Prerequisites
- **Node.js** (v14 or higher)
- **npm** (Node Package Manager)
- **MySQL** Server

### Installation Guide

1. **Install Node.js and npm**
   ```bash

   npm --version
